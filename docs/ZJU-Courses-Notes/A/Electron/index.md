# 电工电子学

## 课程简介
---
教师: 孙晖 学分: $4.5$

成绩构成:

- SPOC 课程 + **学在浙大**作业. **SPOC 只做章节测试, 不用关注视频!!!!**  $20\%$
- 思政报告(你对课程的感受) $5\%$
- 阶段性测试 $3$ 次 $15\%$
- 课堂内作业 $10\%$
- 期末卷面成绩 $50\%$

!!! note
	本门课程和电工电子学实验关联十分密切, 基本上是讲完理论就进行实验操作, 因此实验课的复习资料同样整理在此. 博主的老师是赵江萍, 这门课不同老师的期末随堂验收风格很不一样, 因此其他老师的仅供参考.

!!! danger
	这是一门针对非电类同学(据说还要给其他更多专业的同学?)而言压缩到了极致, 涵盖电类专业同学电路基础, 模拟电路, 数电基础, 电机学基础, 电气控制技术基础, 电工学基础, 一门课仅复习提纲就需要几十页的逆天课程.
	
	博主非常不建议大家速成, 并且实际上几乎也不可能速成. 许多学长学姐即使平时上课认真听讲, 考前通宵复习好几天依旧无济于事. **如果不想让这门课成为大学挂掉的第一门课, 博主非常认真地建议大家平时上课一定要弄清楚主干知识的原理, 千万不要只记住了结论!!!!**
	
	另外, 前面提到的实验课程是一门五级制的课, 并且延续了浙大糟糕的实验报告传统, 平时不但要花很多时间在撰写实验报告上, 而且最后还很可能因为各种情况得到一个不太理想的成绩. 总之, 请上这两门课的同学做好心理准备.
