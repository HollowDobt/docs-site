    Chap I 静力学§

    Abstract

    在静力学一章中, 我们主要研究刚体的三个问题: (1) 受力分析; (2) 力系的等效与简化; (3) 力系平衡建立的条件. 本章的所有内容都是在解决这三个问题, 这也是解本章问题最可靠的三个思路.

    1. 静力学中的受力分析§

    1.1 基本定义§

    • 研究对象 刚体或者可刚化的变形体.
    • 刚体的定义 指在力的作用下, 其内部任意两点之间的距离始终保持不变的物体. 这是一个典型的理想化物理模型.
    • 变形体的定义 指在力的作用下其变形不能忽略的物体(或者说"不满足刚体定义的物体").

    1.2 (静力学)公理体系§

    1.2.1 与变形体兼容的公理§

    • 公理 1 平行四边形法则 作用在同一点上的多个力可以根据平行四边形简化为一个合力, 即:\(\sum\overrightarrow {F_i} =\overrightarrow {F_R}\), 其中\(\overrightarrow{F_R}\)是等效的合力,\(\overrightarrow {F_i}\)是作用在点上的第\(i\)个力.
    • 公理 4 作用力与反作用力 作用力与反作用力同时存在, 并且二个力: (1) 大小相等; (2) 方向相反; (3) 沿着同一直线; (4) 分别作用在两个不同物体上.

    1.2.2 仅作用于刚体的公理与推论§

    • 公理 2 二力平衡条件 作用在同一刚体上的两个力要平衡, 则这两个力: (1) 大小相等; (2) 方向相反; (3) 沿着同一直线. (当然可以将"作用在同一刚体上"作为第(4)点与公理 4 形成对比)
    • 公理 3 加减平衡力系原理 在任一原有力系上加上或者减去任意平衡力系, 与原力系对刚体的作用效果相同.

    针对公理 3, 我们可以进一步推论得到(证明略):

    • 推论 1 力的可传性 力可以在其所在的刚体上沿着其作用线滑移并且不改变作用效果.
    • 推论 2 平面条件下的三力平衡汇交定理 若刚体可以在三个力的共同作用下保持平衡, 那么三个力的作用线必定交于一点, 并且三个力处在同一平面.

    1.2.3 连接刚体与变形体的公理§

    • 公理 5 刚化原理 若变形体在某一力系作用下保持平衡, 那么将变形体刚化为刚体, 仍然能保持平衡.

    1.3 常见的几类约束§

    下面截取自张文普老师的 ppt. 这里博主额外(相对于教学提前)补充了固定端约束的情况.

    • 固定端约束 在平面力系中, 此类约束不但可能会产生\(x\)和\(y\)两个方向的力, 而且也可能会产生一个阻碍刚体旋转的力偶. 我们在后面列平衡方程时需要特别注意.

    1.4 二力构件§

    • 二力构件 指构件上只有两个外力作用点, 并且不考虑自重, 分布力, 内力等附加因素的理想构件.

    根据后面学习的平面力系知识, 对于任意刚体, 要使之保持静止, 必须满足合力与合力矩均为\(0\):

    \[ \sum_{i=1}^n\overrightarrow {F_i}=0\ \ \ \ \ \sum_{i=1}^n\overrightarrow {M_i}=0 \]

    因此, 在平衡条件时, 二力构件所受的两个力必定大小相等, 方向相反, 保持共线, 并且这条线必须沿着构件的形心连线.

    1.5 受力分析习题§

    此类问题解题步骤很固定, 只需要用好前面的知识和尽可能使用平面汇交定理就行. 下面这些问题都是第九版哈工大理论力学的教材习题, 因此这里不给出具体解答. 另外, 下面有部分题的力系非常复杂, 仅用三力汇交是无法确定的(涉及力偶等知识). 此类问题需要分别假设\(\overrightarrow{F_x}\)与\(\overrightarrow{F_y}\)两个分量, 不能直接解出合力的方向.

    2. 平面力系§

    Note

    若力系中各力的作用线处于同一平面内, 那么我们称此力系为平面力系. 平面力系具体分为: 平面汇交力系, 平面力偶系, 平面平行力系, 以及前三种力系混合组成的平面任意力系.

    2.1 基本定义§

    • 力矩 指"力对点之矩", 在哈工大第九版教材中定义为一个有"正负的代数量(标量)", 这是因为在平面力系中力矩只能是垂直于作用平面的方向, 本来也就只有两个方向. 为了方便引入空间力系, 这里给出力矩的标准定义:\(\overrightarrow {M(O)_i}=\overrightarrow {r_i}\times \overrightarrow {F_i}\). 其中\(\overrightarrow {M(O)_i}\)表示力\(\overrightarrow{F_i}\)对\(O\)的矩, \(\overrightarrow {r_i}\)表示\(O\)到力作用点的位矢.
    • 力偶 指刚体上大小相等, 方向相反, 平行但不共线的一对不可再简化的力. 其特点是: (1)构成的力矩大小与选取的矩心无关; (2)力偶只能由力偶平衡, 不能由单一力平衡.

    2.2 基本结论§

    • 平面汇交力系 所有力的作用线交于一点. 此类力系的最终化简结果是一个作用在汇交点的合力:\(\sum\overrightarrow {F_i}=\overrightarrow {F_R}\). 其平衡条件显然是:\(\overrightarrow {F_R} = 0\).
    • 平面力偶系 只受到力偶的作用. 此类力系最终化简结果是一个力偶:\(\sum\overrightarrow {M_i}=\overrightarrow {M_R}\). 其平衡条件显然是:\(\overrightarrow {M_R} = 0\).
    • 平面任意力系 既受到力的作用也受到力偶的作用. 此类力系的最终化简结果是一个合力(主矢)和一个主矩(注意区别合力偶与主矩. 合力偶只是各个力偶之和, 而主矩则需要在合力偶的基础上加上\(\sum\overrightarrow{r_i}\times\overrightarrow{F_i}\)). 设简化中心为\(O\), 则有:\(\sum\overrightarrow {F_i}=\overrightarrow {F_R}\)且\(\sum\overrightarrow {M(O)_i}=\overrightarrow {M(O)_R}\). 其平衡条件显然是:\(\overrightarrow {F_R} = 0\)且\(\overrightarrow {M(O)_R} = 0\).

    Note

    将简化中心从\(O\)改到\(O'\), 很容易证明前后主矢不变. 而主矩则由主矢和两简化中心之间的相对位矢决定, 因为\(\overrightarrow {M(O')_R} = \sum{(\overrightarrow{r_{O'O}} + \overrightarrow{r_{OP}})}\times\overrightarrow{F_i}\), \(\overrightarrow {M(O)_R} = \sum{\overrightarrow{r_{OP}}}\times\overrightarrow{F_i}\). 因此满足:

    \[ \overrightarrow {M(O')_R} = \overrightarrow{M(O)_R} -\overrightarrow{r_{OO'}} \times \overrightarrow{F_R} \]

    从这个表达式可以进一步推导出, 对平面力偶系而言, 刚体上任意一点作为简化中心, 其合力偶都是不变的.

    2.3 静力平衡分析思路§

    接下来讨论确定刚体静力平衡需要满足的约束条件. 因为平面任意力系兼容前几种力系, 因此其约束条件能约束所有种类的平面力系, 求解所有的平面力系平衡条件, 我们只需要掌握这一种约束条件. 而我们知道平面上可以将力投影到\(x\),\(y\)两个方向并且达成同等效果的约束, 加上合力偶只有\(\pm\)之分, 因此得到标量化的求解方程组:

    \[ \sum_{i=1}^n{F_x} = 0\ \ \ \ \ \sum_{i=1}^n{F_y} = 0\ \ \ \ \ \sum_{i=1}^n{M_i} = 0\ \ \ \ \ \]

    上面这种平衡条件称为"二力一矩", 实际计算中"一力二矩", "三矩"的形式都是等价可行的. 下面简单证明.

    2.3.1 一力二矩§

    • 定义 所谓"二矩", 指分别选择两个简化中心简化力矩, 合力矩都为\(0\). "一力", 指选择一个方向(作用线)简化力, 合力为\(0\), 并且这个方向不垂直于两个简化中心连成的直线的作用线.
    • 证明 假定两个力矩简化中心\(A\), \(B\), 已知\(M(A)=0\), \(M(B)=0\), 我们知道单个合力矩方程无法导出有约束的合力方程, 那么可以考虑\(\overrightarrow {M(O')_R} = \overrightarrow{M(O)_R} -\overrightarrow{r_{OO'}} \times \overrightarrow{F_R}\), 带入数据, 即:\({M(B)} = {M(A)} -{r_{AB}} \times {F_R}\)(其中\(F_R\)垂直于\(r_{AB}\)). 又因为\(M(A)=M(B)=0\), 可以确定垂直于\(AB\)方向的合力为\(0\). 再加上额外补充的一个不垂直于\(AB\)方向的合力方程, 其值为\(0\), 因此原式等效为二力一矩方程, 可以确定刚体的静力平衡.

    2.3.2 三矩§

    • 定义 与上面类似, 即选择三个简化中心简化力矩, 合力矩都为\(0\), 并且三个简化中心必须构成一个三角形.
    • 证明 类似地, 单个合力矩方程无法导出有约束的合力方程. 设三个简化中心\(A\), \(B\), \(C\), 根据\(\overrightarrow {M(O')_R} = \overrightarrow{M(O)_R} -\overrightarrow{r_{OO'}} \times \overrightarrow{F_R}\)确定\({M(B)} = {M(A)} -{r_{AB}} \times {F_R}\)与\({M(C)} = {M(A)} -{r_{AC}} \times {F_L}\). 已知三个简化中心构成一个三角形, 则\(\angle BAC\ne 0\), 自然\(F_R\)与\(F_L\)不共线. 根据平面向量基底原理, \(F_R\)与\(F_L\)不共线意味着两个方向力的分量皆为\(0\), 等效于二力一矩方程, 因此同样可以确定刚体的静力平衡.

    2.4 理论力学可解性§

    如上面所述, 若某一物体系由\(n\)个物体组成, 那么可以形成不超过\(3n\)个独立约束方程. 之所以是"不超过", 因为其中部分刚体所受力系是平面汇交力系或者平面平行力系, 前者计算出的合力偶始终为\(0\), 后者则只能写出一个独立的合力方程, 因此约束方程减少为\(2\)个. 根据独立约束方程的数目和未知量数目之间的关系, 我们将问题分为:

    • 静定 未知量数目等于或少于独立约束方程的数目, 可以直接利用理论力学静力学方程求解.
    • 超静定 未知量数目多余独立约束方程的数目, 需要材料力学或结构力学补充方程求解.

    2.5 平面力系例题§

    此类问题的解题步骤同样十分固定. 先分析整体的受力情况, 确定所有(对分析物体的)未知外部约束力, 列出二力一矩或者其他平衡方程, 然后进一步进行局部分析, 根据实际需要列出局部补充方程, 尽可能不要引入未知数, 或者尽可能保证引入未知数的数目少于方程数目(因此力矩方程在局部分析中十分受欢迎).

    此类题目的求解过程十分繁琐, 因此参考答案将只列出平衡方程.

    2.5.1 EX1 结论: 力微元积分等效结果§

    水平梁\(AB\)受三角形分布荷载的作用, 如下图所示. 已知其中的最大值为\(q\), 且梁长度为\(l\), 试求合力的大小及合理作用线的位置.

    Success

    利用好等效力矩的结论: \(F\cdot x_C=\int_0^l{(\frac{q}{l}x\,\mathrm dx)\cdot x}=\frac{1}{3}ql^2\). 又因为:\(F=\int_0^l\frac{q}{l}x\,\mathrm dx=\frac{1}{2}ql\), 联立解得:

    \[ x_C=\frac{2}{3}l \]

    此结论在后面的受力分析中经常使用, 因此将不再过多说明.

    2.5.2 EX2 仅使用整体法求解§

    下图中\(AB\)是杆, \(BC\)是钢索, \(A\)端为铰链约束, 已知\(P_1=10\mathrm{kN}\), \(4P_2=4\text{kN}\), \(\theta=30^\circ\). 求钢索\(BC\)和铰链\(A\)的约束反力.

    Success

    对整体分析, 总计有三个未知量: \(A\)处铰链产生的\(x\), \(y\)方向的力的约束, \(BC\)产生的二力杆件类约束. 对整体列出\(x\), \(y\)方向受力平衡方程和力矩平衡方程即可:

    \[ \left\{\begin{aligned} &\sum F_x=0\longrightarrow F_{A_x}=F_B\cos\theta \\ &\sum F_y=0\longrightarrow P_1=F_{A_y}+F_B\sin \theta \\ &\sum M(A)=0\longrightarrow F_B\sin \theta\cdot |AB|=P_1\cdot |AD| \end{aligned}\right. \]

    2.5.3 EX3 整体法+局部补充方程求解§

    下图是由\(AC\)杆与\(CD\)杆构成的物体系, 其中\(A\)处固结\(AC\)杆, \(AC\)与\(CD\)通过\(C\)处销钉连接. 求解\(A\)对\(AC\)的约束力与\(B\)对\(CD\)的约束力.

    Success

    对整体分析, 总计有四个未知量: \(A\)处产生的\(x\), \(y\)方向的力的约束与约束力偶, \(B\)产生的竖直向上的约束力.仅列出整体的三个方程是不够的, 但是可以先列出, 如果不引入其他未知量就只需要额外一个补充方程:

    \[ \left\{\begin{aligned} &\sum F_x=0\longrightarrow F_{A_x}=0 \\ &\sum F_y=0\longrightarrow F_{A_y}+F_B=P_1 + p\cdot 1+P_2+\frac{1}{2}\cdot q\cdot 1 \\ &\sum M(A)=0\longrightarrow M_A-0.5\times P_1-\frac{3}{2}\times (p\cdot 1) -2.5\times P_2 +3\times F_B - \frac{10}{3}\times (\frac{1}{2}q\cdot 1) = 0 \end{aligned}\right. \]

    我们知道, 唯一可能不引入其他未知量的, 还想要得到补充方程的方法有且仅有力矩方程. 因为\(CD\)杆受力情况相对简单, 我们研究\(CD\)杆部分的受力情况. \(C\)处显然不会提供额外的力矩, 因此让\(CD\)对\(C\)处列力矩平衡方程, 得到补充方程:

    \[ -0.5\times P_2 + 1\times F_B -\frac{4}{3}\times (\frac{1}{2}q\cdot 1) = 0 \]

    四个方程四个未知数, 可解.

    2.5.4 EX4 整体法+局部补充方程求解(Homework)§

    约束与受力情况如图所示, 物体系由\(AB\), \(BC\), \(CD\)三个独立杆件构成. 已知\(AB\)在\(A\)点固结, \(AB\)与\(BC\)在\(B\)点通过销钉连接, \(BC\)与\(CD\)在\(C\)点通过销钉连接, \(CD\)在\(D\)点通过铰链连接. 求钉销\(B\)对\(BA\)的约束

    Success

    对整体分析, 总计有五个未知量: \(A\)点处\(x\), \(y\)方向的约束力和一个约束力偶, \(D\)处\(x\), \(y\)方向的约束力. 那么我们只需要先列出整体的三个受力分析方程, 再列\(AB\)这一部分对\(B\)的力矩平衡方程, 然后列\(CD\)对\(C\)的力矩平衡方程. 这样就有五个独立的平衡方程, 即可解出五个未知量. 接下来对\(AB\)列出\(x\)和\(y\)方向的平衡方程即可求解出\(B\)对\(AB\)的约束.

    注意, 必须要保证列出的两个方程之间独立. 具体而言, 比如上面题目中我们分析了\(AD\)这一整体, 那么就不能分别研究\(AB\)和\(BD\)这两个部分, 因为这两个部分的方程可以等效合成出\(AD\)这一整体的方程. 选用\(AB\)和\(CD\)是因为不用引入除了原有的未知量外其他的未知量.

    2.5.5 EX5 整体法+(选好)局部补充方程求解§

    Success

    4. 空间力系§

    Note

    力系中各力的作用线不处于同一平面内时, 这样的力系称为空间力系. 空间力系中同样分为空间汇交力系, 空间力偶系, 空间平行力系以及前三者混合组成的空间任意力系.

    因为定义几乎完全一样, 这里不再过多赘述这些概念定义.

    4.1 基本定义§

    • 力矩矢 空间中力对点的矩, 定义为: \(\overrightarrow{M(O)}=\overrightarrow{r_i}\times\overrightarrow{F_i}\). 可以利用下面的行列式计算. 但是注意, 这个力矩矢的大小在非纯力偶系下显然是会随着选定的简化中心改变而改变的, 这称为"定位矢量".
    \[ \overrightarrow{M(O)}=\overrightarrow{r_i}\times\overrightarrow{F_i}=(x_i,y_i,z_i)\times (F_x,F_y,F_z)=\begin{vmatrix} i & j & k\\ x & y & z\\ F_x & F_y & F_z \end{vmatrix} \]
    • 力对轴的矩 设力的作用点到轴的距离为\(\overrightarrow r\)(方向: 从轴到力的作用点), 那么对应值为: \(\overrightarrow{M(X)}=\overrightarrow{r_i}\times\overrightarrow{F_i}\).
    • 力矩矢与力对轴的矩的关系 力矩矢对某一轴的分量等于力对该轴的矩. 具体举例而言: \(\overrightarrow{M(O)_x}=\overrightarrow{M(X)}\).
    • 力偶矩矢 就是力偶, 因为在空间力系中方向不是只凭借\(\pm\)能够表示的, 因此添加"矢"字强调这是一个矢量. 另外, 在空间中, 力偶只能在其作用的刚体同一表面上移动. 实例见下图所示.

    • 力螺旋 由一力与一力偶组成的力系, 并且力垂直于力偶作用的平面. 我们区分左螺旋和右螺旋. 区分标准: 若力与力偶形成右手螺旋, 那么是右螺旋; 否则是左螺旋.

    4.2 基本结论(空间任意力系简化结果)§

    • 一个合力偶 没什么好说的, \(\overrightarrow{F_R}=0\), \(\overrightarrow{M}\ne0\).
    • 一个合力 也没什么好说的, \(\overrightarrow{F_R}=0\).
    • 主矢&主矩混合 也就是所谓的\(\overrightarrow{F_R}\ne0\)且\(\overrightarrow{M}\ne0\)分为几种情况. (1)简化的主矩与主矢垂直: 可以将简化中心切换到另一点, 只剩下一个主矢; (2)简化的主矩与主矢平行: 不能再做简化, 构成一个力螺旋; (3)既不平行也不垂直: 最终简化结果同样一定是力螺旋. 示例如下, 最终只留下了与\(\overrightarrow{F_R}\)平行的一个主矩分量\(\overrightarrow{M_R(y)}\)

    4.3 基本结论(空间任意力系平衡方程)§

    平衡条件(基本):

    \[ \sum_{i=1}^n{F_x} = 0\ \ \ \ \ \sum_{i=1}^n{F_y} = 0\ \ \ \ \ \sum_{i=1}^n{F_z} = 0\ \ \ \ \ \sum_{i=1}^n{M_x} = 0\ \ \ \ \ \sum_{i=1}^n{M_y} = 0\ \ \ \ \ \sum_{i=1}^n{M_z} = 0\ \ \ \ \ \]

    当然, 还可以衍生出"四矩二力"等等解法, 但是因为空间力系本身求解的繁琐性, 一般根本用不到这么复杂的结论. 关键是记住, 空间物体系中单个物体至多可以形成六个约束方程(当然可能少于这个量, 原因同样是非任意力系形成的方程少于六个).

    Note

    空间汇交力系, 空间力偶系, 空间平行力系这另外三种力系可提供的约束方程数目都是三个.
